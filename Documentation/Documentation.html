Document for ATM Simulator Application !

This document’s aim is to describe how to install this project .

Main features of the system:

  • Validation – Validate customer card and PIN customer enterred.
  • Withdraw – Allow customer to withdraw money.
  • Check balance – Allow customer to check their account balance.
  • View history – Allow customer to view their succesful transaction.
  • Cash Transfer – Allow customer to transfer their cash to another account in accepted bank system.
  • Change PIN – Allow customer to change their PINt.
  • Logging – System write log.

 

Requirements:

  1. Windows XP or Higher version (Win10 was recommended)
  2. Microsoft .Net Framework v4.0 ++
  3. Microsoft Visual Studio 2019(recommended)
  4. Microsoft SQL Server

 

Installation:

  • Open Project in Visual Studio
  • Import file database in SQL Server 
  • Open Project\Pls\App.config at Solution Explorer to config SQL connection string
  • Run (F5)
